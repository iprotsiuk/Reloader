## Milestone Plan [v0.1/demo]

### Milestone M0 - Runtime Foundation (Completed)

Completed outcomes:
- World scene topology and bootstrap entry flow established.
- MainTown <-> IndoorRange round-trip travel implemented and regression-tested.
- Core persistence slice for inventory/weapons/world-object state implemented.
- Basic inventory and trade flow wiring operational in runtime architecture.

Primary evidence:
- `RoundTripTravelPlayModeTests`
- `WorldObjectStateSaveModuleTests`
- `EconomyControllerCheckoutPlayModeTests`

### Milestone M1 - Demo Core Loop (Current Target)

Target outcome:
- Player can acquire components, perform single-stage bench flow, fire at indoor range, and receive clear shooting result feedback.

Exit criteria:
- Single-stage press interaction has explicit end-to-end PlayMode acceptance coverage.
- Group-size result feedback has acceptance coverage and can move to `Implemented` once evidence is added to this status board.
- Save/load restores the demo-critical loop state without manual repair steps.
- MainTown authored scene defaults (vendor, pickups, travel triggers) pass smoke checks.

### Milestone M2 - Demo Hardening (Next)

Target outcome:
- Stable candidate for external demo sessions.

Exit criteria:
- Regression suites for travel, inventory, persistence, weapons, and trade run clean on the release branch.
- Docs and plans are synchronized: no checklist says "planned" for already-shipped work.
- Demo script (start -> shop -> reload -> shoot -> save/load -> continue) is reproducible without editor intervention.
